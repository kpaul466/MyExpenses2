# Expense Manager - User Manual

Welcome to Expense Manager! This application helps you easily track expenses, maintain shopping lists, and sync everything securely to your own Google Drive.

## 1. Getting Started
When you open the app, you will be on the main Dashboard. 
- **Add Expense:** Tap the big `+` button at the bottom center to add a new transaction.
- **Lock Screen:** You can set up a PIN code to lock your app from casual snooping.

## 2. Navigation
The bottom navigation bar allows you to switch between main views:
- **Dashboard:** Overview of your balance, income, expenses, and recent transactions.
- **Planner:** Create and manage shopping lists securely.
- **Settings:** Configure app preferences, backup, and lock screen.

## 3. Managing Transactions
- Tap the `+` button to open the transaction form.
- You can select if it's an **Income** or **Expense**.
- Enter the amount, choose a category, and optionally add a note.
- In the Dashboard, you can tap on categories to view detailed transactions within them.
- Swipe or tap the trash icon to delete transactions.

## 4. Google Drive Sync (Multi-Device)
To use the app on multiple devices (like your phone and laptop), you can connect to Google Drive:
1. Go to **Settings** (gear icon).
2. Tap the **Google Drive Sync** / Login button.
3. Sign in with your Google Account.
4. Your data is now backed up securely to a hidden app data folder in your Google Drive.
5. On another device, log in with the **same Google Account** to download and merge your data automatically.

## 5. App Installation (PWA)
Expense Manager works offline and can be installed as an App on your phone:
- **iOS:** Tap the Share button in Safari and select "Add to Home Screen".
- **Android:** Tap the 3-dots menu in Chrome and select "Install app".
- **Desktop:** Click the install icon in the URL bar.
